<h2>Aplicativo de gerenciamento para um zoológico</h2>

<p>Este aplicativo Angular permite gerenciar animais e seus cuidados em um zoológico. Ele inclui funcionalidades para listar, adicionar, editar e excluir animais, bem como gerenciar os cuidados associados a cada animal.</p>
<p>O aplicativo também possui uma página de "Não Encontrado" para lidar com rotas inválidas ou recursos inexistentes, melhorando a experiência do usuário ao navegar pelo sistema.</p>


<h2>Tecnologias Utilizadas</h2>
<ul>
  <li>Angular</li>
  <li>TypeScript</li>
  <li>HTML/CSS</li>
  <li>ASP.NET Core (Backend)</li>
  <li>SQL Server (Banco de Dados)</li>
</ul>
