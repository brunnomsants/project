<h2>Aplicativo de gerenciamento para um zoológico</h2>

<p>Este aplicativo Angular permite gerenciar animais e seus cuidados em um zoológico. Ele inclui funcionalidades para listar, adicionar, editar e excluir animais, bem como gerenciar os cuidados associados a cada animal.</p>
<p>O aplicativo também possui uma página de "Não Encontrado" para lidar com rotas inválidas ou recursos inexistentes, melhorando a experiência do usuário ao navegar pelo sistema.</p>


<h2>Tecnologias Utilizadas</h2>
<ul>
  <li>Angular</li>
  <li>TypeScript</li>
  <li>HTML/CSS</li>
  <li>ASP.NET Core (Backend)</li>
  <li>SQL Server (Banco de Dados)</li>
</ul>

<h2>Como rodar o projeto</h2>
<ol>
  <li>Clone o repositório: <code>git clone &lt;URL do repositório&gt;</code></li>
    <li>Navegue até o diretório do projeto: <code>cd ZooApp</code></li>
    <li>Instale as dependências do Angular (caso ainda não tenha): <code>npm install</code></li>
    <li>Inicie o servidor de desenvolvimento Angular: <code>npm start</code></li>
    <li>Abra o navegador e acesse: <code>http://localhost:4200</code></li>
    <li>Para o backend ASP.NET Core, navegue até o diretório do backend e execute: <code>dotnet run</code></li>
    <li>Certifique-se de que o banco de dados SQL Server esteja configurado corretamente e acessível.</li>
</ol>
